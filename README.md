# 파인로드 조편성 앱

4개 조에 이름을 배정하고, 카드처럼 끌어서 순서를 바꿀 수 있는 간단한 칸반 앱입니다.

## 기능

- 이름을 한 명 또는 여러 명씩 추가
- 4개 조 자동 균등 배정
- 조 사이 이동 및 같은 조 안 상하 순서 변경
- 맨 위는 리딩, 맨 아래는 후미 아이콘 표시
- 브라우저 자동 저장

## Railway 배포

Railway에서 이 `group-kanban` 폴더를 프로젝트 루트로 잡아 배포하면 됩니다.

Start command:

```bash
npm start
```

앱은 Railway가 제공하는 `PORT` 값을 사용해 실행됩니다.
